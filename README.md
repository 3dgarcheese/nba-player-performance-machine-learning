# NBA Player Performance Machine Learning

A machine learning project developed in R that analyzes NBA player statistics using supervised learning techniques to classify and predict player performance.

## Overview

This project demonstrates the complete machine learning workflow including data collection, preprocessing, model training, validation, visualization, and performance evaluation.

## Features

- NBA statistics data collection
- Data preprocessing
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Cross Validation
- Model evaluation
- Statistical visualization

## Technologies

- R
- RStudio

## Machine Learning Methods

- Logistic Regression
- KNN Classification
- Cross Validation
- Feature Selection
- Model Evaluation
